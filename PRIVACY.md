# Privacy Policy

## Gemini Tagger

**Last Updated:** March 2026

### Data Collection

Gemini Tagger does **not** collect, transmit, or share any personal data.

### Local Storage

All data stored by this extension is kept **locally** in your browser using the Firefox `storage.local` API. This includes:

- Conversation tags (stored by chat ID)
- User preferences

This data remains on your device and is never transmitted to external servers.

### Third-Party Access

This extension does not:
- Send data to any external servers
- Use analytics or tracking
- Include third-party scripts or services
- Access data from websites other than `gemini.google.com`

### Permissions

The extension requires only one permission:
- **storage**: Used to persist your tags locally in the browser

### Changes to Policy

If this privacy policy changes, the updated version will be posted with a new "Last Updated" date. However, no substantive changes to data practices are anticipated.
